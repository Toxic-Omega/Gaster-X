# Gaster-X

### Install On Linux:
```
git clone https://github.com/Toxic-Omega/Gaster-X
cd Gaster-X
cp Gaster-X.php /var/www/html
cd /var/www/html
php -S 5555
Then Go To "127.0.0.1:5555/Gaster-X.php" And Login With Password "456745"
```
